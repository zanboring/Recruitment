# 招聘数据可视化系统 - 启动命令

## 📋 前提条件

- 确保已安装以下环境：
  - Java 21+
  - Maven 3.6+
  - Node.js 18+
  - MySQL 8.0+
  - Ollama（可选，用于本地AI模型）

- 确保MySQL服务已启动，数据库配置正确：
  - 数据库名：`recruitment_db`
  - 用户名：`root`
  - 密码：`123456`

## 🔧 启动命令

### 方式一：快速启动（推荐）

打开 **PowerShell**，依次执行以下命令：

```powershell
# 1. 关闭旧服务（如果正在运行）
Get-NetTCPConnection -LocalPort 8080,5173 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force -ErrorAction SilentlyContinue }

# 2. 设置环境变量（启用Ollama本地模型）
$env:OLLAMA_ENABLED="true"
$env:OLLAMA_BASE_URL="http://localhost:11434"
$env:OLLAMA_MODEL="qwen2:7b"

# 3. 启动后端服务（新开终端执行）
cd recruitment-system
mvn spring-boot:run

# 4. 启动前端服务（另开终端执行）
cd recruitment-system-frontend
npm run dev
```

### 方式二：分步启动

#### 第一步：启动后端服务

```powershell
# 进入后端目录
cd recruitment-system

# 设置环境变量
$env:OLLAMA_ENABLED="true"
$env:OLLAMA_BASE_URL="http://localhost:11434"
$env:OLLAMA_MODEL="qwen2:7b"

# 编译并启动
mvn clean compile
mvn spring-boot:run
```

**启动成功标志**：
```
Started RecruitmentSystemApplication in X seconds
```

#### 第二步：启动前端服务

```powershell
# 进入前端目录
cd recruitment-system-frontend

# 安装依赖（首次运行需要）
npm install

# 启动开发服务器
npm run dev
```

**启动成功标志**：
```
Local:   http://localhost:5173/
```

## ⚡ 一键启动脚本

创建 `start.ps1` 文件，内容如下：

```powershell
@echo off
Write-Host "=== 招聘数据可视化系统启动脚本 ===" -ForegroundColor Cyan

# 关闭旧服务
Write-Host "`n[1/4] 关闭旧服务..." -ForegroundColor Yellow
Get-NetTCPConnection -LocalPort 8080,5173 -ErrorAction SilentlyContinue | ForEach-Object {
    Write-Host "  关闭端口 $($_.LocalPort)..."
    Stop-Process -Id $_.OwningProcess -Force -ErrorAction SilentlyContinue
}

# 设置环境变量
Write-Host "`n[2/4] 设置环境变量..." -ForegroundColor Yellow
$env:OLLAMA_ENABLED="true"
$env:OLLAMA_BASE_URL="http://localhost:11434"
$env:OLLAMA_MODEL="qwen2:7b"

# 启动后端
Write-Host "`n[3/4] 启动后端服务..." -ForegroundColor Yellow
Start-Process powershell -ArgumentList "-Command", "cd recruitment-system; mvn spring-boot:run" -WorkingDirectory $PWD

# 启动前端
Write-Host "`n[4/4] 启动前端服务..." -ForegroundColor Yellow
Start-Process powershell -ArgumentList "-Command", "cd recruitment-system-frontend; npm run dev" -WorkingDirectory $PWD

Write-Host "`n=== 服务启动完成！===" -ForegroundColor Green
Write-Host "  前端地址: http://localhost:5173"
Write-Host "  后端地址: http://localhost:8080"
```

运行方式：
```powershell
.\start.ps1
```

```powershell
# 停止后端服务（端口8080）
Get-NetTCPConnection -LocalPort 8080 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }

# 停止前端服务（端口5173）
Get-NetTCPConnection -LocalPort 5173 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }
```

## 📝 注意事项

1. **首次运行**：需要先安装依赖
   ```powershell
   cd recruitment-system-frontend
   npm install
   ```

2. **数据库连接**：确保MySQL服务已启动，且数据库配置正确

3. **Ollama配置**：如果不需要本地模型，可以设置 `$env:OLLAMA_ENABLED="false"`

4. **端口冲突**：如果端口被占用，请检查并释放对应端口
