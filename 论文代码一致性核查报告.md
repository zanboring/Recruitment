| 论文描述 | 代码实际实现 | 状态 |
|----------|--------------|------|
| SHA-256哈希生成Job_Key | ✅ **已确认使用** | CrawlServiceImpl.java 使用 `MessageDigest.getInstance("SHA-256")` |

**核查证据**（CrawlServiceImpl.java）：
```java
private String generateJobKey(String url, String title, String company) {
    String raw = url + "|" + title + "|" + company;
    MessageDigest digest = MessageDigest.getInstance("SHA-256");
    byte[] hash = digest.digest(raw.getBytes(StandardCharsets.UTF_8));
    StringBuilder hexString = new StringBuilder();
    for (byte b : hash) {
        String hex = Integer.toHexString(0xff & b);
        if (hex.length() == 1) hexString.append('0');
        hexString.append(hex);
    }
    return hexString.toString();
}
```

**核查结论**：✅ 一致，SHA-256实现正确。
